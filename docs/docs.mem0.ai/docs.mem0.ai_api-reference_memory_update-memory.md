The updated text content of the memory
[​](https://docs.mem0.ai/api-reference/memory/update-memory#response-user-id)

[​](https://docs.mem0.ai/api-reference/memory/update-memory#response-agent-id)
The agent ID associated with the memory, if any
[​](https://docs.mem0.ai/api-reference/memory/update-memory#response-app-id)
The app ID associated with the memory, if any

Hash of the memory content

Additional metadata associated with the memory
